Detailed Description

This is "Logger" class.

It basically prints the messages to all the streams in "streams" vector if the object is "isReady" and "isActive" isReady represents the state if the logger is initialized isActive represents the state if logger should work right now. isActive = false prevents logger::push method from working and doesn't throw any exceptions Both isActive and isReady are false by default and must be modified via methods activate and start correspondingly
